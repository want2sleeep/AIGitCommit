# 截图和演示说明

## 截图要求

VSCode 市场截图规格：
- **格式**: PNG 或 JPG
- **最大宽度**: 1280px（推荐）
- **最小宽度**: 640px
- **宽高比**: 16:9 或 4:3
- **数量**: 建议 3-5 张
- **文件大小**: 每张 < 1MB

## 拍摄技巧

### 环境准备

1. **使用干净的项目**
   - 创建一个示例项目
   - 准备好示例代码变更
   - 确保变更有代表性

2. **VSCode 设置**
   - 使用默认主题或流行主题（如 Dark+）
   - 关闭不相关的侧边栏
   - 调整字体大小，确保清晰可读
   - 隐藏个人信息（用户名、路径等）

3. **屏幕设置**
   - 使用合适的分辨率（1920x1080 或 1280x720）
   - 确保界面元素清晰
   - 避免过小的文字

### 拍摄步骤

1. **准备示例代码**
```typescript
// 示例 1: 添加新功能
export function calculateTotal(items: Item[]): number {
  return items.reduce((sum, item) => sum + item.price, 0);
}

// 示例 2: 修复 Bug
- if (user.age > 18) {
+ if (user.age >= 18) {
    allowAccess();
  }

// 示例 3: 重构代码
- function getUserName(user) {
-   return user.firstName + ' ' + user.lastName;
- }
+ const getUserName = (user: User): string => 
+   `${user.firstName} ${user.lastName}`;
```

2. **录制 GIF**
   - 放慢操作速度
   - 在关键步骤停顿 1-2 秒
   - 确保鼠标移动流畅
   - 避免快速切换

3. **后期处理**
   - 裁剪不必要的部分
   - 添加标注和箭头
   - 优化文件大小
   - 添加说明文字

### 标注工具

推荐使用以下工具添加标注：

- **Snagit**: 专业的截图和标注工具
- **Skitch**: 简单易用的标注工具
- **Photoshop**: 专业图像编辑
- **GIMP**: 免费的图像编辑工具
- **Figma**: 在线设计工具

### 标注样式

- **箭头**: 使用明亮的颜色（红色、橙色）
- **框框**: 使用半透明的高亮框
- **文字**: 使用清晰的字体，足够大
- **阴影**: 添加阴影增加可读性

## 市场页面使用

在 VSCode 市场发布时，你需要：

1. 上传图标 (`icon.png`)
2. 上传截图（按顺序）
3. 可选：上传 GIF 演示

市场会自动生成缩略图和预览。

## 注意事项

1. **隐私保护**
   - 不要显示真实的 API 密钥
   - 不要显示个人项目路径
   - 不要显示敏感代码

2. **质量控制**
   - 确保图片清晰
   - 避免模糊和失真
   - 使用一致的风格

3. **文件大小**
   - 优化图片大小
   - 使用适当的压缩
   - 确保加载速度快

4. **可访问性**
   - 使用高对比度
   - 确保文字可读
   - 添加替代文本描述
